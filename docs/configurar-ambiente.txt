Caso seja necessário instalar Node e NPM, aí vão as instruções.

Windows
=======
Baixar e executar o instalador: http://nodejs.org/

Ubuntu
======
Node
----
http://nodejs.org/dist/v0.10.28/node-v0.10.28.tar.gz
(ou baixar versão mais nova)

tar zxvf node-v0.10.28.tar.gz
cd node-v0.10.28
sudo make
sudo mv out/Release /usr/bin/node

NPM
---
sudo apt-get install curl
curl -L https://npmjs.org/install.sh | sudo sh


Comandos úteis
==============

npm init                         # cria package.json
npm install pacote --save-dev    # baixa pacote e acrescenta dependência ao package.json
npm install                      # baixa as dependências configuradas no package.json
